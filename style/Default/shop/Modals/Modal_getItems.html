Получение предметов в игре

Чтобы получить купленные предметы в игре, воспользуйтесь следующими командами:

/cart - Вывести список купленных вещей

/cart get [кол-во] - Получить [кол-во] вещей с заданным номером, если не указано [кол-во], достаются все вещи c этим ID

/cart all - Получить все купленные вещи

В случае, если нужное количество вещей не помещается в инвентарь, из корзины достается столько вещей, сколько влезет в инвентарь.
